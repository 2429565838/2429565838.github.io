本地图片转URL地址

发表于2024-04-11|更新于2024-04-11
|字数总计:328|阅读时长:1分钟|阅读量:

转换方法

方法一：

在GitHub中获取图片的 HTTPS URL 地址，你可以按照以下步骤进行操作：

  1. 上传图片到GitHub仓库: 将你的图片文件上传到GitHub仓库中。你可以通过在仓库中创建一个新的文件夹，然后将图片文件拖拽到该文件夹中来完成上传。
  2. 定位图片文件: 在GitHub仓库中找到你上传的图片文件。你可以通过在GitHub网站上导航到相应的仓库和文件夹来找到它。
  3. 获取图片的HTTPS URL地址: 找到你的图片文件后，点击它以打开预览。然后，右键点击图片并选择“复制图片地址”或类似的选项（选项名称可能会根据你的浏览器而有所不同）。这将复制图片的 HTTPS URL 地址到剪贴板中。
  4. 使用HTTPS URL: 现在，你可以在你的应用程序、网站或任何需要使用该图片的地方，将刚刚复制的 HTTPS URL 地址粘贴进去使用了。

方法二：

​ 可以把图片上传到一个支持 HTTPS 协议的网络服务器，并获取到该图片的 URL 地址。

​ 例如：微信，QQ，微博等，但是不稳定，最稳定的就是方法一。
